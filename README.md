\# Senado RAG Dashboard



Aplicación en Streamlit para explorar proyectos de ley y plenarias del Senado mediante búsqueda semántica y visualización interactiva.



\## Requisitos

\- Python 3.10 o 3.11



\## Instalación

```bash

python -m venv .venv

.\\.venv\\Scripts\\activate

python -m pip install --upgrade pip

python -m pip install -r requirements.txt
